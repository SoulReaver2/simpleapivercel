SimpleAPI is a basic API using Vercel cloud functions to receive email, source, userAgent, timestamp via POST and store on mongo atlas.

You must suscribe and get your bearer token to use the premium version of the API.

That said, you can still test our simpleAPI and have fun. By default, you are logged in as guest and use the bearer token provided below.

As a guest user, you are limited to 1 call every 30 seconds, but you can tests all the API capabilities: add, delete and export as CSV . For heavier usage, please log in with a premium account.
